Descripción del Proyecto
Página Web destinada a usuarios que deseen tener un registro de cursos, alumnos, profesores y trabajos entregables.


Los usuarios pueden realizar las siguientes accciones:

Cargar en distintos formularios cursos, alumnos, profesores y trabajos entregables.
Visualizar los datos cargados de cursos, alumnos, profesores y trabajos entregables.
Buscar dentro de la base de datos cursos, alumnos, profesores y trabajos entregables.
Ademas de contar en caso de requerirlo, un superusuario administrador.

Para buscar una comision se debe hacer click en el boton INICIO en la esquina superior izquierda, aparecerá un campo donde se ingresa la comision a buscar y un boton de búsqueda.
Para ingresar a los formularios de cursos, estudiantes, profesores y entregables se debe hacer click a los botones azules CURSOS, ESTUDIANTES, PROFESORES, ENTREGABLES respectivamente, los cuales se encuentran en el margen superior.
Para buscar estudiantes se debe poner en la url: AppCoder/buscarEstudiantes.
Para buscar profesores se debe poner en la url: AppCoder/buscarProfesores.
Para buscar entregables se debe poner en la url: AppCoder/buscarEntregables.
